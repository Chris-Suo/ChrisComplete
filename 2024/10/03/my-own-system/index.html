建造一台自己的電腦系列

開發環境

MAC

硬體

HW
bootloader + os + c + browser + database + game

QEMU

BIOS

Basic Input Output System
在real mode下執行
BIOS services:
print characters
disk services
memory map
video mode

Power on self test (POST)執行自檢
找到0x7c00並轉跳，這個位置是MBR (master boot record)

三種模式：
real mode: boot process
protected mode: prepare for long mode
long mode:
包含兩個子模式：64-bit mode, compatibility mode (執行16-bit or 32-bit)

現代電腦使用UEFI (unified extensible firmware interface)試圖取代BIOS

read mode:
讀取kernel
取得硬體資訊

有4個特別的16-bits reg (segment registers)
cs: code segment
ds: data segment
es: extra segment
ss stack segment

address format:
segment reg: offeset(logical address)
segment reg X 16(shift left 4 bits)+offset = physical address (20 bits)

example:
mov ax,[ds:50h] ds=0x1000
0x1000*16+0x50 = 0x10050

pop bx ss=0x100 sp=0x500
0x100*16+0x500 = 0x1500

General purpose registers:
8 bits: al ah bl bh
16 bits: ax bx cx dx
32 bits: eax ebx ecx edx
64 bits: rax rbx rcx rdx (not available in read mode)
